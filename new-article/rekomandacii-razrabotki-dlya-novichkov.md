### Пустая ссылка

Получать пустую ссылку документа лучше НЕ через:

```
Справочники.Сотрудники.ПустаяСсылка()
```

А через предопределённое значение:

```
ПредопределенноеЗначение("Справочник.Сотрудники.ПустаяСсылка")
```

Тоже самое с определением предопределённых элементов справочников или при работе с перечислениями

### Получить значение реквизита

Если нужно получить реквизит какого-то значения (например, получить Контрагента из Договора), то следует писать НЕ так:

```
Договор.Контрагент;
```

А через метод из модуля «ОбщегоНазначения»:

```
ОбщегоНазначения.ЗначениеРеквизитаОбъекта(Договор, "Контрагент")
```

### Получить данные регистров без запроса

Данные регистров (срезы первых и последних) можно получить и без запроса. Каждый регистр имеет предопределённые методы.

-  Регистры сведений (пример МестоположениеОС)

#### Получить срез первых:

```
НачалоПериода = Новый Граница(НачалоПериода, ВидГраницы.Включая)
Отбор = Новый Структура; //в эту структуру нужно передать Измерения, по которым получаем значения
РегистрыСведений.МестонахождениеОС.ПолучитьПервое(НачалоПериода, Отбор)
```
